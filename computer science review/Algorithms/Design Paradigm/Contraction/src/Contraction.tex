\question{Steps}

\begin{itemize}
  \item contract: "contract" (map) instance of problem p to smaller instance
  \item solve: solve smaller instance recursively
  \item expand: use the solution to solve original instance
\end{itemize}

\question{Notes}

\begin{itemize}
  \item inductive proof
  \item work by recursive (inductive) relation
  \item efficient if reduce the problem size geometrically (constant factor $\textless$ 1)
\end{itemize}
