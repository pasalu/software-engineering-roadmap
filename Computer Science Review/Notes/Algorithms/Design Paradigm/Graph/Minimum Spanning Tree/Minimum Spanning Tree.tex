\question{Boruvkas}

Algorithm for minimum spanning tree (smallest weight subgraph with all vertices) of a graph

O(Elog V) where E=edges, v=vertices in the graph

\begin{itemize}
\item Find min edge for all vertices
\item Connect those edges
\item Loop until all connected
\begin{itemize}
   \item Find min edge out of all trees (connected vertices)
   \item Connect those edges
\end{itemize}
\end{itemize}

O(Elog V) where E=edges, v=vertices in the graph
