\question{Administrative}

\begin{itemize}
  \item small informative variables
  \item comments in function definition, if, while, for, etc.
  \item pre/post conditions, loop invariant
  \item input/output types
  \item test null and all possible inputs (and outputs)
  \item initialize a variables you get from else where as null then get them (just in case there's an issue getting them) and test if null
\end{itemize}

\question{Proving Code}

\begin{itemize}
  \item Preconditions, postconditions, assertions, loop invariants
  \item connect algorithmic ideas to imperative programs
  \item proofs + contracts = correctness and safety of code
\end{itemize}
